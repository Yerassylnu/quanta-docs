# Troubleshooting

## 1. Не сохраняется файл

Проверьте:
1. Открыт ли Workspace.
2. Есть ли права на запись в папку.
3. Не удален ли файл/папка во внешнем файловом менеджере.

Быстрый тест:
1. Нажмите `Ctrl/Cmd + S`.
2. Если нужно сохранить в другое место — `Ctrl/Cmd + Shift + S`.

## 2. Поиск не видит только что добавленный текст

Что сделать:
1. Подождать ~1 секунду после ввода (автосохранение/индексация).
2. Закрыть и снова открыть Search (`Ctrl/Cmd + K`).
3. Проверить, что файл имеет расширение `.md` или `.txt` для контент-поиска.

Примечание:
- Поиск по имени файлов работает для всех типов,
- Поиск по содержимому — для текстовых форматов.

## 3. После переименования отображается старое имя

Обычно это решается автоматически через обновление дерева.

Если визуально осталась старая нода:
1. Переключите вкладку (`Editor` -> `System Map` -> назад),
2. или закройте/откройте текущую папку в дереве.

## 4. Не открываются ссылки из Help/About

Проверьте:
1. Что приложение запущено в обычном режиме ОС (не в жесткой песочнице).
2. Что системный браузер доступен.

## 5. Не создается файл/папка

Проверьте имя:
- нельзя пустое,
- нельзя `.` и `..`,
- нельзя использовать `/` и `\\` внутри имени.

## 6. Analytics пустой

Проверьте формат трекеров в файле:
- `$label = current / target`
- пример: `$reading = 12 / 30`

Требования:
- файл должен быть `.md` или `.txt`.

## 7. System Map не показывает wiki-связи

Проверьте синтаксис:
- `[[ИмяФайла]]` или `[[ИмяБезРасширения]]`

Также:
- целевой файл должен существовать в текущем Workspace.

## 8. Как быстро сделать "чистый" перезапуск рабочей сессии

1. Нажмите кнопку выхода в Sidebar (`Back to Hub`).
2. Откройте Workspace снова через `Open Existing`.
3. Это полностью пересоберет дерево и индекс.
