# Hotkeys

Ниже перечислены горячие клавиши, поддерживаемые в текущем приложении.

Примечание:
- `Ctrl` для Windows/Linux.
- `Cmd` (`⌘`) для macOS.
- В таблице используется формат `Ctrl/Cmd`.

## Глобальные

- `Ctrl/Cmd + K` — открыть/закрыть Search.
- `Ctrl/Cmd + O` — открыть папку (`Open Folder`).
- `Ctrl/Cmd + S` — сохранить текущий файл (`Save`).
- `Ctrl/Cmd + Shift + S` — сохранить как (`Save As`).
- `Ctrl/Cmd + N` — создать новый файл.
- `Ctrl/Cmd + Shift + N` — создать новую папку.
- `Ctrl/Cmd + ,` — открыть Settings.
- `F1` — открыть Settings (если фокус не в поле ввода).
- `Esc` — закрыть Search/Settings/About и выйти из inline-режимов.

## Переключение режимов

- `Ctrl/Cmd + 1` — `Editor`.
- `Ctrl/Cmd + 2` — `System Map`.
- `Ctrl/Cmd + 3` — `Analytics`.

## Search (Command Palette)

Когда Search открыт:
- `Arrow Down` — следующий результат.
- `Arrow Up` — предыдущий результат.
- `Enter` — открыть выбранный результат.
- `Esc` — закрыть Search.

## Inline-операции в дереве файлов

Для rename/create input:
- `Enter` — подтвердить.
- `Esc` — отменить.
- `Blur` (клик мимо) — тоже подтверждает действие при валидном имени.
