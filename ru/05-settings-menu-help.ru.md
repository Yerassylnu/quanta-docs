# Settings, Menu Bar, Help

## Settings

Открыть можно:
- `Ctrl/Cmd + ,`
- `F1`
- Sidebar -> `Settings`

Разделы:

### Appearance
- `Theme`:
- `Quanta Dark`
- `High Contrast`
- `Nord (R&D)`
- `Interface Zoom` (80%–120%).

### Editor
- `Auto-save`
- `Focus Mode`
- `Line Numbers`

### License
- Вкладка под Founders Edition.
- Поле ключа лицензии (UI-уровень).

## Верхнее нативное меню (desktop/macOS style)

QUANTA использует нативный menu bar (как в macOS-приложениях).

### File
- `Open Folder...`
- `New File`
- `New Folder`
- `Save`
- `Save As...`
- `Close Window`
- `Quit`

### Edit
- `Undo`
- `Redo`
- `Cut`
- `Copy`
- `Paste`
- `Select All`

### Selection
- `Search`

### View
- `Editor`
- `System Map`
- `Analytics`

### Help
- `About Product`
- `Quanta Website`
- `Documentation`
- `YerassylSystems Website`

## About Product

В модальном окне `About Product` содержится:
- название продукта,
- версия,
- описание,
- переходы на официальные ссылки.

Официальные ссылки:
- Company: `https://yerassylsystems.com`
- Quanta website: `https://quanta-by-yerassylsystems.vercel.app/`
