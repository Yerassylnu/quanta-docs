# Quick Start

## 1. Первый запуск

После запуска QUANTA вы попадаете на `Dashboard`.

Доступные действия:
- `Create New Universe` — создать новый Workspace (проект).
- `Open Existing` — открыть существующую папку как Workspace.
- `Recent Locations` — список последних открытых Workspace.

## 2. Создание нового Workspace

Шаги:
1. Нажмите `Create New Universe`.
2. Укажите имя проекта (`Universe Name`).
3. Выберите папку размещения (`Location`).
4. Выберите шаблон (`Structure Template`).
5. Нажмите `Ignite Universe`.

Поддерживаемые шаблоны в текущей версии:
- `Empty Void` — базовый проект.
- `Second Brain` — структура PARA (`0_Inbox`, `1_Projects`, `2_Areas`, `3_Resources`, `4_Archive`).
- `Product Launch` — папки `Roadmap` и `Sprints`.
- `Content Creator` — отображается в UI, но сейчас создается как базовый проект с `README.md`.

## 3. Открытие существующей папки

1. Нажмите `Open Existing`.
2. Выберите папку в системном диалоге.
3. Папка откроется как текущий Workspace.

## 4. Основной рабочий цикл

1. В левой панели выберите папку/файл.
2. Создайте новый файл (`+`) или новую папку (`folder+`).
3. Введите название и нажмите `Enter`.
4. Для нового файла редактор открывается сразу, курсор ставится в конец текста автоматически.
5. Пишите текст — автосохранение работает автоматически.

## 5. Базовая навигация

Режимы работы:
- `Editor` — редактирование заметок.
- `System Map` — граф структуры папок/файлов и wiki-ссылок.
- `Analytics` — трекеры прогресса (`$label = current / target`).

Переключение:
- через Sidebar,
- через верхнее меню,
- или горячими клавишами (см. `02-hotkeys.md`).
