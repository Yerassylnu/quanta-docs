# Workspace, Files, Folders

## Концепция Workspace

`Workspace` — это корневая папка проекта, с которой вы работаете в текущей сессии.

Особенности:
- Одновременно активен один Workspace.
- Приложение хранит список недавних Workspace (`Recent Locations`).
- Workspace можно удалить из списка recent без удаления файлов с диска.

## Выбор объекта: только один активный

В QUANTA реализован single selection:
- если выбран файл, активен только файл;
- если выбрана папка, активна только папка.

Это важно для операций создания.

## Логика создания нового файла/папки

Когда вы нажимаете `New File` или `New Folder`, цель создается так:
1. Если выделена папка: создать внутри этой папки.
2. Если выделен файл: создать в директории этого файла.
3. Иначе: создать в корне Workspace.

## Создание файла

При создании:
- если расширение не указано, добавляется `.md`;
- в файл записывается стартовый заголовок `# ИмяФайла`;
- файл открывается автоматически в редакторе;
- курсор автоматически ставится в конец текста (можно сразу печатать).

## Переименование

Поддерживается rename для файлов и папок.

Что происходит:
- имя обновляется мгновенно в дереве;
- после успешного rename обновляется путь;
- если был открыт файл, путь активного файла тоже обновляется;
- если была выбрана папка, ее выделение переносится на новый путь;
- дерево/индекс синхронизируются в реальном времени.

Ограничения имени:
- нельзя использовать путь целиком (только имя),
- нельзя использовать `.` и `..`,
- нельзя использовать `/` или `\` в имени.

## Контекстное меню (правый клик)

Для файла/папки доступны:
- `Rename`
- `Copy`
- `Cut`
- `Delete`
- для папки дополнительно:
- `New File Inside`
- `New Folder Inside`
- `Paste` (если в буфере есть объект)

Для пустого места в дереве:
- `Paste Here`
- `New File`
- `New Folder`

## Drag and Drop

Поддерживается перетаскивание файла/папки в папку:
- выполняется перемещение (`move`),
- структура Workspace обновляется.

## Автосохранение и ручное сохранение

- При редактировании контента автосохранение происходит автоматически.
- Можно сохранить вручную: `Ctrl/Cmd + S`.
- `Save As` создаёт копию в выбранный путь и делает ее активной.

## Draft / Inbox

Приложение поддерживает встроенный черновик (`draft.md`) в директории `Documents/Quanta`.

Сценарий:
- пишете в Draft,
- нажимаете `Save Note`,
- заметка переносится в текущий Workspace.
