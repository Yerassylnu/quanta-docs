# Hotkeys

Below are all hotkeys currently supported by the app.

Notes:
- `Ctrl` on Windows/Linux.
- `Cmd` (`⌘`) on macOS.
- This document uses `Ctrl/Cmd` notation.

## Global

- `Ctrl/Cmd + K` — open/close Search.
- `Ctrl/Cmd + O` — open folder (`Open Folder`).
- `Ctrl/Cmd + S` — save active file (`Save`).
- `Ctrl/Cmd + Shift + S` — `Save As`.
- `Ctrl/Cmd + N` — create new file.
- `Ctrl/Cmd + Shift + N` — create new folder.
- `Ctrl/Cmd + ,` — open Settings.
- `F1` — open Settings (when focus is not in an input/editor field).
- `Esc` — close Search/Settings/About and exit inline edit modes.

## View Switching

- `Ctrl/Cmd + 1` — `Editor`.
- `Ctrl/Cmd + 2` — `System Map`.
- `Ctrl/Cmd + 3` — `Analytics`.

## Search (Command Palette)

When Search is open:
- `Arrow Down` — next result.
- `Arrow Up` — previous result.
- `Enter` — open selected result.
- `Esc` — close Search.

## Inline File Tree Actions

For rename/create input:
- `Enter` — confirm.
- `Esc` — cancel.
- `Blur` (click outside) — also confirms if input is valid.
