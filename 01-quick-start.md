# Quick Start

## 1. First Launch

After launching QUANTA, you land on the `Dashboard`.

Available actions:
- `Create New Universe` — create a new Workspace (project).
- `Open Existing` — open an existing folder as Workspace.
- `Recent Locations` — list of recently opened Workspaces.

## 2. Create a New Workspace

Steps:
1. Click `Create New Universe`.
2. Enter project name (`Universe Name`).
3. Select location (`Location`).
4. Choose a template (`Structure Template`).
5. Click `Ignite Universe`.

Supported templates in current version:
- `Empty Void` — basic project.
- `Second Brain` — PARA structure (`0_Inbox`, `1_Projects`, `2_Areas`, `3_Resources`, `4_Archive`).
- `Product Launch` — creates `Roadmap` and `Sprints`.
- `Content Creator` — available in UI, currently created as a basic project with `README.md`.

## 3. Open an Existing Folder

1. Click `Open Existing`.
2. Select folder in system dialog.
3. The folder opens as active Workspace.

## 4. Main Workflow

1. Select a folder/file in the left sidebar.
2. Create a new file (`+`) or folder (`folder+`).
3. Type a name and press `Enter`.
4. New files open automatically in the editor.
5. Start typing right away; autosave runs automatically.

## 5. Navigation

Work modes:
- `Editor` — note editing.
- `System Map` — graph view of files/folders and wiki-links.
- `Analytics` — tracker dashboard (`$label = current / target`).

You can switch via:
- Sidebar,
- top native menu,
- or hotkeys (see `02-hotkeys.md`).
