# Settings, Menu Bar, Help

## Settings

Open Settings via:
- `Ctrl/Cmd + ,`
- `F1`
- Sidebar -> `Settings`

Sections:

### Appearance
- `Theme`:
- `Quanta Dark`
- `High Contrast`
- `Nord (R&D)`
- `Interface Zoom` (80%–120%).

### Editor
- `Auto-save`
- `Focus Mode`
- `Line Numbers`

### License
- Founders Edition UI section.
- License key input field (UI-level in current version).

## Native Top Menu (desktop/macOS style)

QUANTA uses a native menu bar similar to standard desktop apps.

### File

- `New File`
- `New Folder`
- `Save`
- `Save As...`
- `Close Window`
- `Quit`

### Edit
- `Undo`
- `Redo`
- `Cut`
- `Copy`
- `Paste`
- `Select All`

### Selection

### View

### Help
